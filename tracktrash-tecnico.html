TrackTrash — Detalhes Técnicos

Documento de especificação técnica: tipo de solução, linguagens/frameworks utilizados e manual completo de uso para Lojistas e Administradores.

Questão 01

Qual é o tipo de solução criada?

Website, aplicativo, SaaS, bot, etc.

✅ Resposta

O TrackTrash é uma aplicação web SaaS (Software as a Service) B2B, acessível diretamente pelo navegador (browser) de qualquer dispositivo — computador, tablet ou smartphone — sem necessidade de instalação. A interface é responsiva, funcionando como um Progressive Web App (PWA), o que significa que o usuário pode adicionar o atalho na tela inicial do celular para uma experiência equivalente a um aplicativo nativo.

🌐
Web App (SaaS)
Acesso via link direto no navegador. Sem instalação, sem App Store.
tracktrash-pitch.vercel.app
📱
Mobile Responsivo
Interface adaptada para telas pequenas. Funciona em qualquer smartphone Android ou iPhone.
🏢
B2B Multi-Tenant
Cada empresa (CNPJ) terá seu próprio ambiente isolado de dados e usuários.
🗂️ Arquitetura Dual-Role — Dois perfis em uma única plataforma
🏪
Portal do Lojista (Cliente B2B)
Dashboard para empresas gerenciarem seus resíduos eletrônicos
Cadastro de resíduos, solicitação de coleta, acompanhamento de status e download de Certificado ESG. Interface simplificada pensada para usuários não-técnicos de pequenas empresas.
⚙️
Admin Dashboard (Operador TrackTrash)
Painel de comando para equipe logística
Módulo SAT-INTEL com mapa interativo em tempo real, feed de inteligência, otimização de rotas por IA (Cérebro Logístico) e gestão de toda a operação de coleta.
Questão 02

Quais linguagens e frameworks foram utilizados?

Se foi Low-Code ou No-Code informe a plataforma.

✅ Resposta

A solução foi desenvolvida utilizando uma combinação de código personalizado (custom code) com práticas de Low-Code para o ambiente Mendix. A stack principal é JavaScript / JSX (React 18) para o front-end, com CSS3 puro para o sistema de design, e Mendix Studio Pro (plataforma Low-Code) para a camada de negócios e banco de dados.

Questão 03

Qual é o manual da solução criada?

Definir quando e como o usuário acessa e utiliza sua solução.

✅ Resposta

O TrackTrash é acessado via navegador web (Chrome, Safari, Firefox ou Edge) em qualquer dispositivo. O endereço único de acesso é https://tracktrash-pitch.vercel.app. Não é necessário instalar nenhum aplicativo. A plataforma possui dois perfis de usuário com jornadas distintas: o Lojista (cliente B2B) e o Administrador (operador TrackTrash). O sistema detecta automaticamente o perfil pelo e-mail informado no login.

🔑 Como Acessar a Plataforma
🔗 Endereço
https://tracktrash-pitch.vercel.app
Funciona em qualquer navegador moderno
🕒 Disponibilidade
24h / 7 dias
Hospedagem cloud na Vercel — sem downtime
Perfil E-mail de Exemplo Senha Acesso a
🏪 Lojista / Empresa loja@empresa.com.br Qualquer senha Portal do Cliente B2B
⚙️ Administrador admin@tracktrash.com Qualquer senha Dashboard de Operações
ℹ️
Lógica de roteamento de perfil

O sistema identifica automaticamente o perfil pelo e-mail: qualquer e-mail contendo a palavra admin direciona para o Dashboard Administrativo. Os demais e-mails direcionam para o Portal do Lojista.

🏪 Jornada do Lojista — Portal B2B (Passo a Passo)
1
Quando usar
Acesso e Login
O lojista acessa o link da plataforma pelo navegador do computador ou celular. Na tela de login, insere o e-mail corporativo e uma senha. O sistema reconhece o perfil e direciona ao Portal do Lojista.
📱 Funciona no celular💻 Funciona no desktop
2
Tela Inicial
Visualizar o Dashboard
Ao entrar, o lojista vê seu painel com os KPIs: total de kg reciclados, número de coletas realizadas e quantidade de Certificados ESG gerados. Também visualiza o histórico recente de coletas com o status de cada uma.
📊 KGs reciclados🏆 Certificados ESG📋 Histórico
3
Cadastro de Resíduo
Registrar um Novo Resíduo Eletrônico
Quando a empresa acumular resíduos eletrônicos (sucata de computador, celulares quebrados, cabos, placas-mãe etc.), o responsável acessa "Meu Inventário" e clica em "+ Cadastrar Resíduo". Preenche: tipo de material (ex: Placa-mãe, Bateria, Celular), quantidade em kg, descrição opcional e localização. Ao salvar, o item aparece no inventário como "Aguardando Coleta".
📦 Tipo de material⚖️ Peso em kg📍 Localização
4
Solicitação de Coleta
Agendar uma Coleta com a TrackTrash
Quando o lojista quiser solicitar a retirada dos resíduos, acessa "Solicitar Coleta". O sistema exibe todos os itens do inventário pendente. O lojista confirma os itens, escolhe uma data preferencial e clica em "Confirmar Agendamento". O sistema exibe o CO₂ que será evitado com esse descarte correto.
📅 Escolhe data🌿 CO₂ evitado✅ 1 clique
5
Acompanhamento
Monitorar o Status da Coleta
Após o agendamento, o lojista pode acompanhar o progresso da coleta na aba de histórico. O status evolui em três etapas: "Pendente" (aguardando confirmação da TrackTrash) → "Em Rota" (caminhão a caminho) → "Coletado" (resíduo retirado com sucesso).
⏳ Pendente🚛 Em Rota✅ Coletado
6
Passo Final
Baixar o Certificado ESG
Quando a coleta chega ao status "Coletado", o sistema libera automaticamente um Certificado Digital de Descarte Responsável. O lojista acessa o "Histórico ESG", localiza a coleta concluída e clica em "Ver Certificado ESG". O documento exibe os dados da empresa, tipo de material, peso coletado, data e o responsável. Pode ser exportado e usado em relatórios de sustentabilidade e auditorias da PNRS.
🏆 Certificado Digital📄 Exportar PDF✅ Conformidade PNRS
⚙️ Jornada do Administrador — Command Center
1
Acesso Admin
Login como Administrador
O operador TrackTrash acessa o mesmo link da plataforma, mas utiliza um e-mail contendo "admin" (ex: admin@tracktrash.com). O sistema reconhece o perfil e redireciona ao Painel de Comando exclusivo.
2
Monitoramento
Visualizar o Feed de Inteligência
A tela inicial do Admin exibe o Feed de Inteligência Logística: solicitações novas de lojistas, alertas de rota, eficiência da frota, KPIs ambientais (toneladas coletadas, CO₂ evitado) e status geral da operação em tempo real.
3
SAT-INTEL
Abrir o Módulo de Rastreamento por Satélite
O Admin clica na aba "SAT-INTEL" na barra lateral. Abre-se um mapa interativo real (OpenStreetMap) com zoom, pan e interatividade total. O mapa exibe marcadores verdes (lojistas com coletas pendentes) e um marcador de caminhão (frota TrackTrash). Ao clicar em qualquer marcador, surge um balão com os detalhes da empresa ou do veículo.
🟢 Pontos de coleta🚛 Frota em tempo real📍 Clique nos pinos
4
Logística
Otimizar Rotas com o Cérebro Logístico (IA)
Na aba "Otimização IA", o sistema agrupa automaticamente as coletas pendentes por proximidade geográfica e sugere a rota otimizada para os caminhões, minimizando km rodado e emissões de CO₂. O Admin confirma o despacho com um clique.
5
Conclusão
Confirmar Coleta e Gerar Certificado
Após o caminhão realizar a coleta física, o Admin marca a coleta como "Coletado" no sistema. Automaticamente, o sistema gera o Certificado ESG para o lojista e atualiza todos os KPIs ambientais da plataforma.
